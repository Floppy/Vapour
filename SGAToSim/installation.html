SGAToSim 1.7

Converter Usage & Avatar Installation Instructions

Usage

The application sgatosim.exe is a Win32 command-line app used to generate self-extracting executables containing models for 'The Sims', created from SGA1 avatars. It is executed from the MS-DOS command prompt in the form:-

sgatosim avatar_filename.ame [sfx_path\]sfx_name.exe {-option argument ...}

No model will be produced unless these options are specified:-

avatar_filename.ameFilename for the input SGA avatar
[sfx_path\]sfx_name.exeOutput filename of the self-extracting executable. The sfx_path\ is optional, and is independent of the -path option

These options may be used to specify a model output directory and properties for the exported model.

-name [model_name]Name to be used for model files - Default: sfx_name
-path [pathname]Directory in which to place temporary model files. The directory sga_temp is created within the executable working directory and used if this option is not specified.
-age (adult|child)Sim age - Default: adult
-sex (male|female)Sim sex - Default: male
-skin (light|medium|dark)Sim skin tone - Default: medium
-build (fit|fat|skinny)Sim build - Default: fit
-0No text output

The default model produced, if no properties are specified, is an adult male of fit build with light skin tone.
This example set of command-line arguments will convert test.ame into a female child with fit build and light skin tone. The temporary model files (*_model.*) will be placed in the directory d:\model_files. These files will then be moved into a self-extracting executable called c:\sfx\test.exe.

sgatosim test.ame c:\sfx\test.exe -name model -path d:\model_files -age child -sex female

N.B. This is the default usage of sgatosim. These command-line arguments may be changed by editing sgatosim.cpp. Please read Source Documentation for details.

Files

The conversion process generates six files. Four model files:

  • cmx3\C999xx_name.cmx
  • cmx3\B999xxxxx_name.cmx
  • cmx3\xskin-C999xx_name-HEAD-HEADB.skn
  • cmx3\xskin-B999xxxxx_name-PELVIS-BODY.skn
And two image files:
  • Skins\C999xxxxx_name.bmp
  • Skins\B999xxxxxxxx_name.bmp

These files are then compressed into the self-extracting executable name.exe.

Installing models into 'The Sims'

The above files need to be installed in your Sims folder. This is normally C:\Program Files\Maxis\The Sims\, but you may have installed to a different place on your computer. Within this folder, there is a folder called GameData. The self-extracting executable can be run from any location on the computer on which you wish to install the model. If the game has been installed correctly, the Sims folder should be found automatically and the files installed into the correct subfolders without any other user intervention. Should the installer not be able to find the Sim folder, you will be prompted to specify the folder yourself.

Installing models into 'SimShow'

The models can also be viewed in SimShow, a model viewer for The Sims. First of all, install the files by running the executable. The files need to be installed in the SimShow folder instead. This is normally C:\Program Files\Maxis\SimShow\, but you may have installed to a different place on your computer. Within this folder, there is a folder called GameData. Within this folder, copy the model files from The Sims\GameData\cmx3 into the SimShow\GameData\cmx3 folder, and the image files from The Sims\GameData\Skins into the folder named SimShow\GameData\Textures. Note that this is a different folder name to the normal Sims installation.

Use

In The Sims, go to the 'Create A Sim' screen, find the appropriate head and body, and start playing! In SimShow, select the correct model attributes, then select the appropriate head and body models and textures. NOTE: Using the 'Install into game' button in SimShow will not work - this only works for textures, not complete models. NOTE: Whilst the game will only allow the correct texture to be used with any given body, SimShow does not. Exported textures used on standard game meshes (or vice versa) will not display correctly.

Problems

I can't find my Sim!
There are a number of reasons this might happen:
  1. Check that you're looking at the right combination of sex, age and skin tone in the Create-A-Sim screen. If you're not, you won't see your Sim.
  2. Check that the files are in the right directories. Make sure the images are in the 'Skins' directory, not the 'Textures' directory (assuming you're using 'The Sims' - for SimShow, it's the other way round).
