SGAToSim

Converter Usage & Avatar Installation Instructions

Usage

The application sgatosim.exe is a Win32 command-line app used to generate models for 'The Sims' from SGA1 avatars. It is executed from the MS-DOS command prompt in the form:-

sgatosim avatar_filename.ame [sim_path\]sim_name.exe {-option argument ...}

No model will be produced unless these options are specified:-

avatar_filename.ameFilename for the input SGA avatar
[sim_path\]sim_name.exeOutput filename of the self-extracting zip file. The sim_path\ is optional, and is independent of the -path option

These options may be used to specify a model output directory and properties for the Sim model.

-path [pathname]Directory in which to place temporary model files. Executable working directory is used if this option is not specified.
-age (adult|child)Sim age - Default: adult
-sex (male|female)Sim sex - Default: male
-skin (light|medium|dark)Sim skin tone - Default: medium
-build (fit|fat|skinny)Sim build - Default: fit
-0No text output

The default model produced, if no properties are specified, is an adult male of fit build with medium skin tone.
This example set of command-line arguments will convert test.ame into a female child with fit build and medium skin tone. The temporary Sims model files (*_test.*) will be placed in the directory d:\temp. These files will then be moved into a self-extracting zip file called c:\Sims\GameData\test.exe.

sgatosim test.ame C:\Sims\GameData\test.exe -path D:\temp -age child -sex female

N.B. This is the default usage of sgatosim. These command-line arguments may be changed by editing sgatosim.cpp. Please read Source Documentation for details.

Files

The conversion process generates six files. Four model files:

  • C999xx_name.cmx
  • B999xxxxx_name.cmx
  • xskin-C999xx_name-HEAD-HEADB.skn
  • xskin-B999xxxxx_name-PELVIS-BODY.skn
And two image files:
  • C999xxxxx_name.bmp
  • B999xxxxxxxx_name.bmp

These files are then compressed into the self-extracting zip file name.exe.

Installing models into 'The Sims'

The above files need to be installed in your Sims folder. This is normally C:\Program Files\Maxis\The Sims\, but you may have installed to a different place on your computer. Within this folder, there is a folder called GameData. Copy the self-extracting model executable into the GameData directory and run with the command-line argument -dir to extract the files into correct directories.

Installing models into 'SimShow'

The models can also be viewed in SimShow, a model viewer for The Sims. First of all, extract the files by running the executable. The files need to be installed in the SimShow folder instead. This is normally C:\Program Files\Maxis\SimShow\, but you may have installed to a different place on your computer. Within this folder, there is a folder called GameData. Within this folder, copy the model files into the cmx3 directory, and the image files into the directory named Textures. Note that this is a different directory name to the normal Sims installation.

Use

In The Sims, go to the 'Create A Sim' screen, find the appropriate head and body, and start playing! In SimShow, select the right filter and then select the appropriate models. NOTE: Using the 'Install into game' button in SimShow will not work - this only works for textures, not complete models.

Problems

I can't find my Sim!
There are a number of reasons this might happen:
  1. Check that you're looking at the right combination of sex, age and skin tone in the Create-A-Sim screen. If you're not, you won't see your Sim.
  2. Check that the files are in the right directories. Make sure the images are in the 'Skins' directory, not the 'Textures' directory (assuming you're using 'The Sims' - for SimShow, it's the other way round).
