SGAToHL 1.1

Converter Usage & Avatar Installation Instructions

Usage

The application sgatohl.exe is a Win32 command-line app used to generate self-extracting executables containing models for 'Half-Life', created from SGA1 avatars. It is executed from the MS-DOS command prompt in the form:-

sgatohl avatar_filename.ame [sfx_path\]sfx_name.exe {-option argument ...}

No model will be produced unless these options are specified:-

avatar_filename.ameFilename for the input SGA avatar
[sfx_path\]sfx_name.exeOutput filename of the self-extracting executable. The sfx_path\ is optional, and is independent of the -path option

These options may be used to specify a model output directory and properties for the exported model.

-name [model_name]Name to be used for model files - Default: sfx_name
-path [pathname]Directory in which to place temporary model files. The directory sga_temp is created within the executable working directory and used if this option is not specified.
-0No text output

This example set of command-line arguments will convert test.ame into a Half-Life model. The temporary model files (model\model.*) will be placed in the directory d:\model_files. These files will then be moved into a self-extracting executable called c:\sfx\test.exe.

sgatohl test.ame c:\sfx\test.exe -name model -path d:\model_files

Files

The conversion process generates two files (using the above example):

  • d:\model_files\name\name.mdl
  • d:\model_files\name\name.bmp

These files are then compressed into the self-extracting executable c:\sfx\test.exe.

Installing models into 'HalfLife'

The above files need to be installed in your game folder. This is normally C:\Sierra\Half-Life, but you may have installed to a different place on your computer. Within this folder, there are the sub-folders valve\models\player. The self-extracting executable can be run from any location on the computer on which you wish to install the model. If the game has been installed correctly, the game folder should be found automatically and the files installed into the correct subfolders without any other user intervention. Should the installer not be able to find the game folder, you will be prompted to specify the folder yourself.

Use

In Half-Life, go to the 'Multiplayer\Customize' screen, then click the 'Next' and 'Previous' buttons in the 'Player Model' window until you find your avatar. The installed avatars are only used within multi-player games.
