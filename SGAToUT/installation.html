SGAToUT 1.0

Converter Usage & Avatar Installation Instructions

Usage

The application sgatout.exe is a Win32 command-line app used to generate self-extracting executables containing models for 'Unreal Tournament', created from SGA1 avatars. It is executed from the MS-DOS command prompt in the form:-

sgatout avatar_filename.ame [sfx_path\]sfx_name.exe {-option argument ...}

No model will be produced unless these options are specified:-

avatar_filename.ameFilename for the input SGA avatar
[sfx_path\]sfx_name.exeOutput filename of the self-extracting executable. The sfx_path\ is optional, and is independent of the -path option

These options may be used to specify a model output directory and properties for the exported model.

-name [model_name]Name to be used for model files - Default: sfx_name
-path [pathname]Directory in which to place temporary model files. The directory sga_temp is created within the executable working directory and used if this option is not specified.
-sex (male|female)Model sex - Default: male
-0No text output

This example set of command-line arguments will convert test.ame into a female Unreal Tournament model. The temporary model files will be placed in the directory d:\model_files. These files will then be moved into a self-extracting executable called c:\sfx\test.exe.

sgatout test.ame c:\sfx\test.exe -name model -path d:\model_files -sex female

Files

The conversion process generates four files (using the above example):

  • d:\model_files\System\name.u
  • d:\model_files\System\name.int
  • d:\model_files\System\nameSkins.int
  • d:\model_files\Textures\name.utx

These files are then compressed into the self-extracting executable c:\sfx\test.exe.

Installing models into 'Unreal Tournament'

The above files need to be installed in your game folder. This is normally C:\UnrealTournament, but you may have installed to a different place on your computer. The self-extracting executable can be run from any location on the computer on which you wish to install the model. If the game has been installed correctly, the game folder should be found automatically and the files installed into the correct subfolders without any other user intervention. Should the installer not be able to find the game folder, you will be prompted to specify the folder yourself.

Playing as your avatar

In Unreal Tournament, go to the 'Options\Player Setup' menu screen, then select your avatar from the 'Class' drop-down menu. The installed avatars are only used within multi-player and practice games.

Playing against your avatar

In Unreal Tournament, start a game with the menu option 'Game\Start Practice Session'. Click on the 'Bots' tab to bring up the bot configuration menu. Click the 'Configure' button to configure individual bots. Select your avatar in the 'Class' drop-down menu for the bot.

To have more that one of your avatar running round, select your avatar for Bot 1,2,3,etc. Uncheck 'Random Order' in the 'Bots' menu screen to make sure the bots are selected in consecutive order.
